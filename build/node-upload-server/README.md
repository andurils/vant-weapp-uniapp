# Node.js server to upload Images

## To run the project

1.Switch to __node-image-upload-server__ directory,
```
cd node-image-upload-server
```
2.Create __images__ directory
```
mkdir images
```
3.Download dependencies
```
npm install
```
4.Run the project
```
node app
```
