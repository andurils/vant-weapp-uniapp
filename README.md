# VantUI Uniapp 示例

本示例原本是从 vant 官方发布的微信演示示例转换而来,支持微信小程序（H5 不支持！！）。 vant-weapp 版本为 1.6.7.

vant weapp 的组件库，都在工程的 wxcomponents 目录下。pages 目录下是各个示例页面。

## 图片上传服务

提供了图片 Nodejs 服务 ，代码路径为\src\node-image-upload-server

安装 npm 包

```
npm install
```

运行服务

```
node app
```
